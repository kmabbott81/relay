TROUBLESHOOTING
---------------

Connection drops after 30s
  -> Add heartbeat every 10 seconds

Duplicates on reconnect
  -> Track event IDs, send last_event_id

Slow reconnect time
  -> Check exponential backoff configuration

High latency
  -> Disable buffering with X-Accel-Buffering: no

Memory leak
  -> Clean up old streams after 1 hour

See SSE_QUICK_REFERENCE.md for full troubleshooting table
